# 펜플로우 (PenFlow) — 문서 인덱스

이 폴더는 네이버 블로그 AI 글쓰기 자동화 도구 **펜플로우(PenFlow)**의 기획·설계·구현 문서를 담는다.
아래 순서대로 읽으면 *무엇을 / 왜 / 어떻게* 만드는지 단계적으로 이해할 수 있다.

## 🔗 문서 간 관계

```
01 시나리오 ──(설계화)──► 02 기능설계서 ──(작업화)──► 03 구현체크리스트
                                                          │
                                       (구현 기술 결정) ◄──┤
                                              ▼            │
                                        04 기술스택        ▼ (착수 단계만 상세)
                                              │     마일스톤/M1-작업분해
                                              │            │
                                              └───────────►│ (스택 선택 반영)
                                                           ▼ (검증)
                                                   마일스톤/M1-테스트케이스
```

> `04-tech-stack`은 `02`의 기능/규칙과 `03`의 구현 항목을 **무엇으로 만들지**로 변환한다(예: 형태소 분석 → Kiwi WASM, 상태머신 → XState). 마일스톤 작업분해는 이 스택을 전제로 진행한다.

> 마일스톤 분해는 **착수할 단계만 상세히, 나머지는 굵게**(rolling-wave) 원칙을 따른다.
> 현재 상세화된 것은 **M1**뿐이며, M2~M5는 `03-implementation-checklist` 부록의 마일스톤 표 수준으로만 둔다. M1 완료 후 M2를 상세 분해한다.

## ⚙️ 범위 메모

- 펜플로우는 **네이버 블로그 단일 시나리오**에 집중한다.
- 멀티 플랫폼 게시·영상/GIF 제작 등은 의도적으로 제외했다(근거는 `02-functional-design` 부록 B 참조).
- 구현 대상 브라우저는 **Chromium 계열 전용**(크롬·네이버 웨일·엣지 등)이다. Firefox·Safari는 제외한다(근거는 `04-tech-stack` 참조).
