# 펜플로우 (PenFlow) — 구현 체크리스트

> 기준 문서: `02-functional-design.md`
> 사용법: 각 항목은 **독립적으로 검증 가능한 완료 기준(DoD)** 을 가진다. `[ ]`를 `[x]`로 바꿔 진행을 추적한다. 괄호 안 `R-x.x`는 설계서 규칙 번호.
> 권장 빌드 순서는 맨 아래 「부록: 마일스톤」 참조.

---

## 0. 공통 기반 (Foundation)

- [ ] **컴포넌트 인터페이스 정의** — ①~⑩ 간 입출력 타입 확정 (설계서 16장 계약표 기준)
- [ ] **마커 규약 확정** — 소제목/이미지/상품표/백링크/광고문구 마커 표기 1곳에 정의 (R-2.4)
  - DoD: 마커 상수 정의 파일 1개, 생성기·비주얼·합성기가 모두 이를 참조
- [ ] **데이터 모델 구현** — Settings/Topic/Visual/Payload/InsertQueue/AutoPostSession (설계서 15장)
- [ ] **로컬 저장소 추상화** — 읽기/쓰기/삭제 인터페이스 (벤더 교체 가능)
- [ ] **로깅·진행상태 채널** — 단계별 진행/실패를 UI·로그 양쪽에 전달 (비기능: 관측성)
- [ ] **에러 처리 표준** — 사용자 통지 + 안전 종료 패턴 공통화

## 1. ① 설정 관리자

- [ ] AI 본문 생성 인증 키 **복수 등록** UI·저장 (R-0.1)
- [ ] 키 **자동 순환**: 한도 초과 감지 → 다음 키 전환, 전부 소진 시 명확한 오류 (R-0.2)
- [ ] AI 모델 선택 + 저장
- [ ] AI 이미지 / 외부 이미지 / 키워드분석 / 제휴 인증 정보 입력 (선택)
- [ ] 인증 정보 **로컬 전용 저장** 보장, 평문 노출·하드코딩 금지 (R-0.3, 비기능: 보안)
- [ ] 설정 **내보내기/불러오기** (파일·클립보드·클라우드) (R-0.4)
- [ ] 저장 시 형식 검증 + 성공/실패 피드백 (6.4)
- [ ] 시계 오차 대비: 서명 인증 실패 시 시간 확인 안내 (R-0.5)
- DoD: 키 1개 소진 시뮬레이션 → 다음 키로 자동 전환되고 로그에 기록됨

## 2. ② 주제 선정기

- [ ] **경로 A** 키워드 분석: 메인 키워드 → 연관/자동완성 수집
- [ ] (선택) 키워드별 검색량·경쟁도 표시
- [ ] 키워드 목록 **일괄 업로드/다운로드**
- [ ] **경로 B** 경쟁 블로그 제목 수집: 블로그 식별자 입력/자동인식 → 개수 지정 → 제목 표
- [ ] 제목 클릭 시 단계 2로 전달
- [ ] **경로 C** 연관 키워드 추출: 다중 소스(네이버·구글·동영상) on/off
- [ ] 경로 C **자동완성 제목 추천** 제공
- [ ] 수집 작업 **중단 가능 + 진행률 표시** (R-1.2)
- [ ] 소스 **어댑터 추상화**(추가/교체 가능) (R-1.3)
- DoD: 세 경로 결과가 모두 동일한 `Topic`(키워드/제목 1건)으로 수렴 (R-1.1)

## 3. ③ 본문 생성기

- [ ] 입력: 주제 키워드 + 프롬프트 (+선택 참고자료)
- [ ] "제목+본문 가져오기" — 현재 페이지 크롤링 → 참고자료화
- [ ] 프롬프트 조립(주제+프롬프트+참고자료)
- [ ] **생성 방식 A** 직접 호출 (키 사용, 빠름)
- [ ] **생성 방식 B** 대화형 서비스 자동화 (키 불필요, 대기시간 설정)
- [ ] 응답 후처리: 코드블록 마크·설명 래퍼 제거 (8.2-3)
- [ ] **마커 포함 보장** — 출력 본문에 약속된 마커가 들어있는지 검사 (R-2.2, R-2.4)
- [ ] 프롬프트 **이름 저장/불러오기/삭제** (R-2.1)
- [ ] 생성 실패 시 재시도/키 전환/사유 통지 (R-2.3)
- [ ] (선택) 키워드별 클릭 시 **일괄 자동 생성** (R-2.5)
- DoD: 생성 결과가 소제목 단위로 구조화되고 마커 검사를 통과

## 4. ⑩ 본문 검증·편집기

- [ ] **형태소 분석**: 본문 분해 → 키워드 횟수·밀도 표 출력 (9.1)
- [ ] 권장 밀도 범위 **설정 가능** + 벗어난 키워드 경고 (R-8.2)
- [ ] **찾기·바꾸기**: 일괄 치환 + 치환 개수 표시 (9.2)
- [ ] **HTML ↔ 마크다운 변환** + 미리보기 + 클립보드 복사 (9.3)
- [ ] 편집/변환 결과가 본문 HTML을 덮어쓰되 **마커 보존** (R-8.3, R-8.4)
- [ ] 단계 전체 **건너뛰기 가능** (R-8.1)
- DoD: 변환 왕복(HTML→MD→HTML) 후에도 마커가 1개도 손실되지 않음

## 5. ⑨ 비주얼 생성기

- [ ] **대표(H1) 썸네일** 생성
- [ ] **소제목(H2) 썸네일**: 소제목 텍스트 합성, 배경(업로드/URL/동영상 캡처) 선택 (10.3)
- [ ] 소제목 썸네일 **소제목 수만큼 1:1 생성·매칭** (R-7.3)
- [ ] **본문 이미지** 생성
- [ ] 이미지 확보 **4모드**: AI생성 / 외부소스 / 업로드(드래그앤드롭) / 기본·클립보드 (10.2, R-7.2)
- [ ] **캐릭터/모델 이미지 일관성**: 참조 이미지 등록·재사용, 생성 시 동반 전송 (10.5, R-7.7)
- [ ] **중복 이미지 회피**: 메타데이터 제거/픽셀 변형 → 매번 고유 바이트 (10.4, R-7.4)
- [ ] 외부 호출 **지연시간(min~max)** 설정으로 rate limit 회피 (R-7.5)
- [ ] 품질(압축) 조절 슬라이더 + 미리보기·일괄삭제·다운로드 (10.6)
- [ ] 생성 비주얼이 본문 마커와 **순서·개수 일치** (R-7.6)
- [ ] 단계 **건너뛰기 가능**(텍스트만 발행) (R-7.1)
- DoD: 소제목 3개 글 → H2 썸네일 정확히 3개, 중복회피 ON 시 모두 다른 바이트, 모델 등록 시 인물 일관

## 6. ④ 부가요소 합성기

- [ ] 본문 마커 순회 → **순서 보장 작업 큐** 생성 (11.3, R-3.3)
- [ ] 쇼핑/제휴 링크: 지정 소제목 위치마다 삽입 (위치 사용자 지정)
- [ ] 상품표 + **최신 데이터 자동 반영** 옵션 (R-3.5)
- [ ] CTA 버튼 / 백링크 블록 / 출처 링크
- [ ] **광고/협찬 문구** 지정 위치 삽입, **옵션 ON이면 누락 금지** (R-3.4)
- [ ] 마커에 대응 리소스 없으면 스킵(단, 광고문구 제외) (R-3.1)
- [ ] 본문 내 순서 = 마커 순서, 임의 재배치 금지 (R-3.2)
- [ ] **비주얼 정합성 검사**: 이미지 마커 수 = Visual 수 (16장 계약2)
- DoD: 정합성 불일치 시 합성 차단 + 사용자 통지

## 7. ⑤ 페이로드 저장소 + ⑥ 삽입 엔진

### ⑤ 페이로드 저장소
- [ ] 본문/이미지/옵션을 삽입 엔진에 전달하는 저장·인출 구현

### ⑥ 삽입 엔진
- [ ] 편집기 준비 **폴링 + 타임아웃** (R-4.1)
- [ ] 초기 팝업 자동 정리
- [ ] 작업 큐 순차 실행: 텍스트/HTML, 이미지(링크부착), 쇼핑링크(불필요 텍스트 제거), 표/백링크
- [ ] 작업 간 **대기시간 조절 가능**(하드코딩 지양) (R-4.3)
- [ ] **제목 추출·입력**
- [ ] **스타일 일괄 적용**(줄간격·글꼴·글자크기) (12.2-5)
- [ ] 포커스 없을 시 **본문 클릭 안내 + 재시도** (R-4.2)
- [ ] 오류 시 **실패 작업 명시 + 안전 종료** (R-4.4)
- [ ] 편집기 모드(팝업/사이드바) **권장 모드 안내** (R-4.5)
- [ ] UI 변경 대비 **폴백 셀렉터** (R-4.1)
- DoD: 텍스트+이미지+표 혼합 글이 순서대로 들어가고, 1개 작업 실패해도 패닉 없이 종료

- [ ] 예약 시각 트리거 발행 (R-6.5)
- [ ] **포스팅 간격(무작위 min~max)** + 위험 감지 시 자동 확대/복구 (R-6.6, 계정 보호)

- DoD: 3개 글 연속 발행 시 간격이 설정 범위 내 무작위로 적용되고, 중단 후 재개가 동작

## 10. 예외/엣지 케이스 (횡단 검증, 설계서 17장)

- [ ] AI 키 한도 초과 → 다음 키 전환/전부 소진 통지
- [ ] AI 빈 응답·형식 오류 → 복구 시도 → 실패 시 재생성/중단
- [ ] 형태소 밀도 과다 → 경고(자동 차단 아님)
- [ ] 이미지 생성 실패 → 대체 모드 폴백 또는 비우고 진행
- [ ] 소제목 수 ≠ 썸네일 수 → 합성 전 차단
- [ ] 중복회피 후 동일 바이트 잔존 → 재변형 1회 → 실패 시 경고
- [ ] 편집기 영역 미발견 → 타임아웃 + 안내
- [ ] 마커 리소스 없음 → 스킵(광고문구 제외)
- [ ] 삽입 중 중단 → 정리 후 종료
- [ ] 로컬 저장 용량 부족 → 미터 경고 + 품질↓ 유도
- [ ] 로컬 시계 오차 → 시간 확인 안내

## 11. 비기능 요구사항 (설계서 18장)

- [ ] **이식성**: AI 벤더·이미지 소스·저장 인터페이스 추상화 확인
- [ ] **복원력**: 재시도·타임아웃·폴백 전 구간 적용
- [ ] **관측성**: 단계별 진행/실패 로그+UI
- [ ] **보안**: 인증 정보 로컬 보관, 평문 노출 없음
- [ ] **저장 용량 관리**: 사용량 미터 + 품질(압축) 조절
- [ ] **정책 준수**: 광고/협찬 표기 보장
- [ ] **계정 보호**: 무작위 포스팅 간격 + 위험 시 자동 확대

## 부록: 권장 마일스톤 (빌드 순서)

| 단계 | 범위 | 목표 |
|---|---|---|
| **M1 — 최소 동작(Walking Skeleton)** | 0 공통 + ① + ③ + ⑤ + ⑥ + ⑦ | 키 저장 → 단일 주제로 텍스트 본문 생성 → 편집기 삽입 → 임시저장 |
