# 네이버 블로그 AI 작성 — M1 테스트 케이스

> 기준: `M1-작업분해.md` (Walking Skeleton)
> 범위: 키 저장 → 주제 입력 → 텍스트 본문 생성 → 편집기 삽입 → 임시저장
> 표기: **TC-영역-번호** / 구분(✅정상 · ❌실패 · ⚠️경계) / `R-x.x`는 설계서 규칙
> **M1 범위 밖**(이미지·표·광고문구·자동화·복수키 순환·발행)은 본 문서에서 테스트하지 않는다.

- **편집기 상태 시드**: ① 정상 ② 로드 지연 ③ 초기 팝업 존재 ④ 미포커스 ⑤ 영역 없음
